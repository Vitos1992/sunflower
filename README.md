# Детский сад «Солнышко» — шаблон лендинга

Файлы:
- index.html — главная страница
- styles.css — стили
- script.js — простая интерактивность

Запуск: откройте `index.html` в браузере (локально). Для быстрой проверки можно использовать Live Server в VS Code или любой статический хост.

Дальше можно:
- заменить заглушечные изображения на свои
- подключить форму к почтовому сервису или backend
- добавить перевод/контент заказчика
